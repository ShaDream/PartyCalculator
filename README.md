# Telegram чат бот для разделения трат на покупки между участниками

Поможет разделить траты между участниками и скажет кто кому и сколько должен перевести.

## Запуск
Для запуска потребуется postgres база данных, с созданными таблицами из database папки.

Так же потребуется токен бота телеграма.

## Функционал
  - Создавать, смотреть, удалять участников
  - Создавать, смотреть, удалять чеки
  - Создавать, редактировать, удалять и использовать группы (алиас для выделения множества участников в чеках)
